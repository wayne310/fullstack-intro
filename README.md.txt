# Full Stack Introduction

## What is Full Stack Development?
Full Stack Development involves both frontend and backend development of a web application.

## Frontend
The frontend is the user interface or visible part of the application.
It uses HTML, CSS, and JavaScript.

## Backend
The backend handles server-side logic, APIs, and database communication.

## Web Application Architecture

User → Browser → Frontend → Backend → Database

The frontend sends requests to the backend.
The backend processes data and communicates with the database.
