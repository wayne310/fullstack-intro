# Mongoose Schemas Lab

## Steps Covered
- Basic Schema Creation
- Data Validation (required, min)
- Default Values
- Schema Options (timestamps, Boolean, Date)
- Subdocuments (Embedded Documents)

## How to Run

Install dependencies:
npm install mongoose

Run each step:

node step1_basicSchema.js
node step2_validation.js
node step3_defaultValues.js
node step4_schemaOptions.js
node step5_subdocuments.js

## Notes
Each step demonstrates schema evolution and validation behavior in MongoDB using Mongoose.
